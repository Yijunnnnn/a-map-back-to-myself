# A Map Back to Myself

**When self-doubt feels true, let your evidence speak.**

A Map Back to Myself is a privacy-first, CBT-informed, evidence-based reflection agent that helps users examine self-doubt through their own personal records instead of receiving generic reassurance.

When a user writes a thought such as:

> “I feel like I haven’t made any progress.”  
> “I can do nothing well.”  
> “I always fail when I try something new.”

the system treats that belief as a hypothesis. It checks the thought against structured personal evidence, identifies possible CBT-related thinking patterns, retrieves relevant evidence cards, and generates a more balanced reflection with citations.

This project is not a therapy system, does not diagnose users, and is not intended for crisis intervention. It focuses on one narrow CBT-inspired workflow:

```text
self-belief → evidence check → balanced thought → small next step
